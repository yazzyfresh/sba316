Reflection Questions

1. What could you have done differently during the planning stages of your project to make the execution easier?

I could have planned my pages and features better before starting. Having a clear idea of the layout and what I wanted each page to do would have made coding easier and saved time.

---

2. Were there any requirements that were difficult to implement? What do you think would make them easier to implement in future projects?

The JavaScript features were the most difficult, especially making the search bar and forms work correctly. In future projects, I can make things easier by planning my code first and testing each feature as I build it.

---

3. What would you add to, or change about your application if given more time?

If I had more time, I would add more reviews, improve the design, and add more interactive features like saving user reviews and adding more search options.

---

4. Notes for My Future Self

I should plan my projects before coding and keep my files organized. I learned that testing often and checking for small mistakes makes debugging easier. I should continue practicing JavaScript and adding new features to improve my skills.
